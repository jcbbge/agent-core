# Nuggets

Atomic facts, CLI one-liners, code snippets, configuration patterns.

## Format

```markdown
# [Name]

**Source:** [optional URL]
**Date:** YYYY-MM-DD
**Tags:** [cli|config|code|pattern]

## The Nugget
```[language if applicable]
[the actual content]
```

## When to Use
[one sentence on when this applies]
```

## Examples

- `fd-find-replace.md` — One-liner for find + sed
- `docker-clean-volumes.md` — Cleanup command
- `solid-memo-deps.md` — Reactive dependency pattern

## Naming

`[name].md` — Keep it short, memorable.

Examples:
- `docker-prune-all.md`
- `git-uncommit-last.md`
- `solid-store-update.md`
