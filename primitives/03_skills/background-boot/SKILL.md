# Background Boot (Alembic Wake)

This skill executes the decoupled wake protocol. It reads the `alembic` substrate for cryptographically verified offline synthesis (`emergence` shards generated by the `alembic-daemon` dream process).

## Procedure

Execute the unified daemon to pull offline synthesis:

```bash
~/agent-core/scripts/alembic-daemon.js wake
```

Read the output. If verified synthesis exists, integrate it into your context for this session. The daemon now outputs `refractions` containing `criticality`, `serendipity`, and `present_collision` scores, along with a `self_critique`.
