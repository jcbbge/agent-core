## DO NOT USE

These converge toward generic AI output:

- **Inter** as display font (acceptable only in `swiss` pairing, not as default)
- **Roboto** in any position
- **Arial** / system fonts as display
- **Space Grotesk** as a default fallback (use only when deliberately chosen)

When `random` is selected: choose from any pairing above with equal probability, weighted slightly toward the mood that matches the user's topic/purpose.
