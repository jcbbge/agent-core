# Style Presets Reference

Complete preset definitions combining font pairings + palettes + signature elements.
These are the pre-composed styles available during Phase 2 style discovery.

For individual font pairings and palettes, see FONT_PAIRINGS.md and PALETTES.md.

---

## Dark Themes

### Bold Signal
**Vibe:** Confident, bold, modern, high-impact
**Pairing:** `signal` — Archivo Black + Space Grotesk
**Palette:** `signal-orange`
**Signature:** Bold colored card as focal point, large section numbers, navigation breadcrumbs, grid layout

### Electric Studio
**Vibe:** Bold, clean, professional, high contrast
**Pairing:** `studio` — Manrope 800 + Manrope 400
**Palette:** `electric-blue`
**Signature:** Two-panel vertical split, accent bar on panel edge, quote typography as hero

### Creative Voltage
**Vibe:** Bold, creative, energetic, retro-modern
**Pairing:** `voltage` — Syne + Space Mono
**Palette:** `voltage`
**Signature:** Electric blue + neon yellow, halftone textures, neon badges

### Dark Botanical
**Vibe:** Elegant, sophisticated, artistic, premium
**Pairing:** `botanical` — Cormorant + IBM Plex Sans
**Palette:** `dark-warm`
**Signature:** Abstract soft gradient circles (blurred, overlapping), warm accents, thin vertical lines, italic signatures

## Light Themes

### Notebook Tabs
**Vibe:** Editorial, organized, elegant, tactile
**Pairing:** `notebook` — Bodoni Moda + DM Sans
**Palette:** Cream paper card on dark: `--bg-outer: #2d2d2d; --bg-page: #f8f6f1`
**Signature:** Paper container with shadow, colorful section tabs on right edge, binder holes

### Pastel Geometry
**Vibe:** Friendly, organized, modern, approachable
**Pairing:** `pastel` — Plus Jakarta Sans
**Palette:** `--bg-primary: #c8d9e6; --card-bg: #faf9f7`
**Signature:** Rounded card with soft shadow, vertical pills on right edge

### Split Pastel
**Vibe:** Playful, modern, friendly, creative
**Pairing:** `split` — Outfit
**Palette:** `--bg-peach: #f5e6dc; --bg-lavender: #e4dff0`
**Signature:** Two-color vertical split, playful badge pills, grid overlay

### Vintage Editorial
**Vibe:** Witty, confident, editorial, personality-driven
**Pairing:** `vintage` — Fraunces + Work Sans
**Palette:** `--bg-cream: #f5f3ee`
**Signature:** Abstract geometric shapes (circle outline + line + dot), bold bordered CTA boxes, conversational copy

## Tufte Lineage (austere)

### Warm Paper
**Vibe:** Austere, scholarly, restrained, museum wall text
**Pairing:** `classic` — Playfair Display + Source Serif 4
**Palette:** `paper`
**Signature:** Hairline rules, kicker with rule draw, field-maps, accent left-ruled code blocks, zero decoration

### Paper & Ink
**Vibe:** Editorial, literary, thoughtful
**Pairing:** `paper-ink` — Cormorant Garamond + Source Serif 4
**Palette:** `cream`
**Signature:** Drop caps, pull quotes, elegant horizontal rules

## Specialty

### Neon Cyber
**Vibe:** Futuristic, techy
**Pairing:** `cyber` — Clash Display + Satoshi (Fontshare)
**Palette:** `dark-navy`
**Signature:** Particle backgrounds, neon glow, grid patterns

### Terminal Green
**Vibe:** Developer-focused, hacker aesthetic
**Pairing:** `terminal` — JetBrains Mono only
**Palette:** `--bg: #0d1117; --accent: #39d353`
**Signature:** Scan lines, blinking cursor, code syntax styling

### Swiss Modern
**Vibe:** Clean, precise, Bauhaus-inspired
**Pairing:** `swiss-modern` — Archivo + Nunito
**Palette:** `--bg: #ffffff; --ink: #000000; --accent: #ff3300`
**Signature:** Visible grid, asymmetric layouts, geometric shapes

## DO NOT USE (Generic AI Patterns)

**Fonts:** Inter as display, Roboto, Arial, system fonts
**Colors:** `#6366f1` generic indigo, purple gradients on white
**Layouts:** Everything centered, identical card grids, generic hero sections
**Decorations:** Realistic illustrations, gratuitous glassmorphism, drop shadows without purpose
