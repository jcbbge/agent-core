# Ideas

Future considerations, things to try, experiments to run.

## Format

```markdown
# [Idea Name]

**Created:** YYYY-MM-DD
**Status:** [backlog|active|done|pass]

## The Idea
[What it is]

## Why
[Why this matters, what problem it solves]

## Next Steps
- [ ] step 1
- [ ] step 2

## Related
- [related idea or reference]
```

## Naming

`[yyyy-mm-dd]-[short-name].md`

Examples:
- `2026-04-17-voice-cli.md`
- `2026-04-15-notion-api-sync.md`
