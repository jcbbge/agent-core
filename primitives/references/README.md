# References

Saved URLs with summaries. Documentation you revisit, cheatsheets, important links.

## Format

```markdown
# [Name]

**URL:** [link]
**Saved:** YYYY-MM-DD
**Tags:** [docs|cheatsheet|api|guide]

## Summary
[2-3 sentences. What is this? Why save it?]

## Key Sections
- [section 1]
- [section 2]
- [section 3]

## When to Reference
[one sentence on when to come back here]
```

## Naming

`[domain]-[topic].md`

Examples:
- `github-actions-secrets.md`
- `mdn-css-grid.md`
- `solid-start-routing.md`
