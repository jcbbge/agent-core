## 3. User Experience Quality (25% weight)

### Cognitive Load Assessment

**Time to first value** (goal: < 5 minutes): ______ minutes

**Number of steps to complete core job:** ______ (fewer is better)

**Concepts user must learn:** ______ (fewer is better)

**Does core value proposition require explanation?** Yes / No

### Usability Checklist

- [ ] System status is visible (users know what's happening)
- [ ] Matches real-world patterns (works like users expect)
- [ ] User has control (can undo mistakes easily)
- [ ] Consistent patterns (predictable interface)
- [ ] Errors prevented (dangerous actions are hard to trigger)

### Flow State Potential

- [ ] Clear goal users work toward
- [ ] Immediate feedback on actions
- [ ] Challenge matches skill level
- [ ] Users can focus without distraction

### Simplicity Test

**Can you explain core value in one sentence?** Yes / No

If yes, write it:
_____________________________________________________

**Can a new user complete core job without tutorial?** Yes / No

### Score: ____ / 10

**Scoring guidance:**
- 8-10: Simple, intuitive, low cognitive load, clear value
- 5-7: Moderate complexity, some learning required
- 1-4: Complex, high friction, requires explanation
