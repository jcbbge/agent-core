# Creative Systems Thinking (Quick Use)

**When:** System design, breaking conventional patterns  
**Trigger:** "think outside the box" | "Ted Nelson" | "10x version"

## The Prompt

```
Think about a 10x version of this system. If you were to build this for yourself, what would be the best possible version optimized for your consumption, use, output?

Think outside the box. Make connections from disparate fields, map ideas from not so obvious links. I want creative Ted Nelson type of ideas. Zig Zag data structures.

In my mind it's like that eye machine at the optometrist - better one, better two. Words, connections, associations, tags, sentences, paragraphs, articles all can have this interconnected webbing that is contextual lenses being swapped out like a phoropter.

I give you permission to think freely. Use creativity.
```

## Key Concepts

**Ted Nelson / Zig Zag:**
- Non-hierarchical organization
- Multi-dimensional connections
- Transclusion (content in multiple contexts)
- Parallel documents

**Phoropter Metaphor:**
- Context as lens/filter
- A/B comparison between framings
- Rapid perspective switching

## 10x Thinking Triggers

- "Build for yourself" → remove abstraction layers
- "Disparate fields" → cross-domain pattern matching
- "Permission to think freely" → break norms

## When to Use

- System architecture rethinking
- Knowledge management systems
- Breaking from conventional patterns
- Long-term strategic thinking
